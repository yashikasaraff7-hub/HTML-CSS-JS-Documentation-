# HTML-CSS-JS-Documentation-
Overview
SkillSwap is a peer-to-peer platform that enables users to swap skills and learn from each other without spending any money. It operates on a credit basis where users earn credits for teaching a skill and then use the credits to learn a different skill.
Problem
Some people cannot access learning platforms due to the fact that most learning platforms require money to access the learning material.
Solution
SkillSwap enables users to teach a skill, earn credits for the taught skill, and then use the credits to learn a different skill from a different user.
Technologies Used
	•	HTML
	•	CSS
	•	JavaScript
Conclusion
SkillSwap enables the creation of a collaborative learning environment where people can share what they know and learn new skills.
